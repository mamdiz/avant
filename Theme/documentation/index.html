1.2 CSS

CSS files – with usage details (order of importance):

  • bootstrap.min.css – Stylesheet (minified) for Bootstraps front-end framework
  • owl.carousel.min.css – Basic stylesheet (minified) for content sliders/carousels
  • font-awesome.css – Font Icons stylesheet
  • animate.css – Stylesheet for CSS3 animations
  • magnific-popup.css – Stylesheet for Image Popups
  • bootstrap-datetimepicker.css – Stylesheet for for Bootstrap Date and Time Selection
  • style.css – Main template stylesheet
  • media.css – Main template Responsive stylesheet

1.3 Javascript

JavaScripts and integrated jQuery scripts – with usage details (order of importance):

  • jquery.min.js – Script (minified) for jQuery script usage
  • bootstrap.min.js – Script (minified) for Bootstraps front-end framework
  • element.js – Main template script
  • modernizr.js – JavaScript library that detects which HTML5 and CSS3 features your visitor's browser supports.
  • owl.carousel.js – jQuery plugin that lets you create beautiful responsive carousel slider.
  • jquery.localscroll-1.2.7-min.js – jQuery plugin that lets Animated anchor navigation made easy with jQuery.
  • wow.min.js – Reveal Animations When You Scroll.
  • imagesloaded.pkgd.min.js – Detect when images have been loaded.
  • jquery.isotope.min.js – Filter & sort magical layouts
  • jquery.parallax-1.1.3.js – The lightest parallax plugin that enhances progressively from basic background images to a nice parallax effect.
  • jquery.magnific-popup.js – Responsive lightbox & dialog script with focus on performance and providing best experience for user with any device
  • moment-with-locales.js – Script for Date/time selection.
  • bootstrap-datetimepicker.js – Date/time picker widget based on twitter bootstrap

3.2 Logo

It is highly recommended to use vector graphics for the logo like the Unicase template does with the SVG (Scalable Vector Graphics) file format. This ensures the best possible and sharpest view on mobile devices and of course when scaling the logo. If it is not possible for you to use vector graphics you can still use raster graphics like the JPG or PNG file format of course. In this case you have to change the filename extension of the logo in the following HTML parts to this for example:

<!-- Logo Section Begins -->
<div class="logo">
    <a class="navbar-brand" href="index.html"><img src="images/logo.png" alt="logo" /></a>
</div>
<!-- Logo Section Ends -->
